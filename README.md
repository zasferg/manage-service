<h1>Проект</h1>
Веб-приложение для управления командой внутри компании. 
<h2>Система позволяет</h2>
1.Регистрироваться и управлять аккаунтами пользователей</br>
2.Создавать команды (компании) и управлять ими</br>
3.Назначать задачи сотрудникам и отслеживать их выполнение</br>
4.Оценивать работу сотрудников</br>
5.Создавать встречи и просматривать календарь событий</br>

<h2>Запуск проекта</h2>
1. Скачайте репозиторий</br>
2. Откройте консоль и перейдие в папку с репозиторием, </br>
далее запустите проект посредством команды, docker  compose up --build </br>
Для сваггера переидите по ссыллке https://127.0.0.1:8000/docs/ </br>
Для запуска админки перейдите по ссылке https://127.0.0.1:8000/admin/ </br>
3. Для запуска миграций в консоли зайдите в docker образ приложения.</br>
Далее перейдите в в папку проекта и запустите alembic --revision autogenerate, alembic upgrade head.
