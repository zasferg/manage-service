# Проект: Веб-приложение для управления командой внутри компании

## Описание

Это веб-приложение предназначено для управления командами, задачами, оценками сотрудников и событиями в рамках одной компании. Проект разработан на FastAPI и использует JWT-аутентификацию. Предоставляет REST API для фронтенда или внешних интеграций.

---

## Возможности системы

- Регистрация и авторизация пользователей
- Управление профилем пользователя
- Создание и управление компаниями и командами
- Назначение задач, изменение их статуса, комментирование
- Проведение встреч и ведение календаря событий
- Оценка сотрудников и просмотр аналитики

---

## Установка и запуск

### Требования

- Docker
- Docker Compose

### Шаги

1. Клонируйте репозиторий:

    ```bash
    git clone git@github.com:zasferg/manage-service.git
    cd pmanage-service
    ```

2. Соберите и запустите проект:

    ```bash
    docker-compose up --build
    ```

3. Доступ к интерфейсам:
   - Swagger-документация: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
   - Админка: [http://127.0.0.1:8000/admin](http://127.0.0.1:8000/admin)

4. Примените миграции:

    ```bash
    # Зайдите в контейнер
    docker exec -it web /bin/bash

    # Примените миграции Alembic
    alembic revision --autogenerate -m "initial"
    alembic upgrade head
    ```

## API: Эндпоинты и описание

### 📌 Аутентификация (`/auth`)

| Метод | URL                | Описание                        | Авторизация |
|-------|--------------------|----------------------------------|-------------|
| POST  | `/auth/registration` | Регистрация нового пользователя | ❌          |
| POST  | `/auth/login`        | Получение JWT токена            | ❌          |
| POST  | `/auth/logout`       | Выход (аннулирование токена)    | ✅          |
| POST  | `/auth/get_access`   | Получение нового access-токена по refresh | ❌ |

---

### 👤 Пользователи (`/users`)

| Метод | URL           | Описание                          | Авторизация |
|-------|---------------|------------------------------------|-------------|
| GET   | `/users/me`   | Получить текущего пользователя     | ✅          |
| PUT   | `/users/me`   | Обновить данные пользователя       | ✅          |
| DELETE| `/users/me`   | Удалить свою учётную запись        | ✅          |

---

### 🏢 Компании (`/companies`)

| Метод | URL                    | Описание                             | Авторизация |
|-------|------------------------|---------------------------------------|-------------|
| POST  | `/companies/create`    | Создание новой компании              | ✅          |
| GET   | `/companies/get?id=`   | Получение информации о компании      | ✅          |
| PUT   | `/companies/add_user`  | Добавить пользователя в компанию     | ✅ (админ)  |
| PUT   | `/companies/delete_user` | Удалить пользователя из компании   | ✅ (админ)  |

---

### ✅ Задачи (`/tasks`)

| Метод | URL                         | Описание                                | Авторизация |
|-------|-----------------------------|------------------------------------------|-------------|
| POST  | `/tasks/task`              | Создание задачи                         | ✅          |
| GET   | `/tasks/task?task_id=`     | Получение задачи по ID                  | ✅          |
| PUT   | `/tasks/task?task_id=`     | Обновление задачи                       | ✅          |
| DELETE| `/tasks/task?task_id=`     | Удаление задачи                         | ✅          |
| PUT   | `/tasks/update_status`     | Обновить статус задачи (`new_status`)   | ✅          |
| POST  | `/tasks/assign_to`         | Назначить задачу пользователю           | ✅          |
| POST  | `/tasks/create_comment`    | Добавить комментарий к задаче           | ✅          |
| GET   | `/tasks/chat_history`      | Получить чат между двумя пользователями по задаче | ✅ |

---
